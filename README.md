# Emeka Okoye | Personal Portfolio

Welcome to my personal portfolio. This website is a single-page application (SPA) built using modern web technologies to showcase my skills, experience, and projects as a Frontend Developer and AI enthusiast.

## Technologies Used

- **HTML5**: Semantic structure for accessibility and SEO.
- **Tailwind CSS**: A utility-first CSS framework for rapid and consistent styling.
- **JavaScript (Vanilla)**: For interactivity, such as the mobile menu and icon rendering.
- **Lucide Icons**: Sleek, modern outline icons.
- **Google Fonts (SF Pro)**: Premium typography for a high-end feel.

## Key Features

- **Single Page Layout**: Smooth navigation between sections using anchor links.
- **Responsive Design**: Optimized for all devices, from mobile to desktop.
- **Glassmorphism**: Modern UI effects using backdrop blurs and subtle borders.
- **Portfolio Gallery**: A visual showcase of projects and personal highlights.
- **Contact Form**: A functional-ready contact interface for potential collaborations.

## Structure

- `index.html`: The main entry point containing all sections (Hero, About, Skills, Experience, Projects, Gallery, Contact).
- `profile.png`: Hero section profile image.
- `imgs/`: Directory for additional assets.

## How to Run

Simply open `index.html` in any modern web browser. No build process is required as dependencies are loaded via CDN.
