# Emeka Okoye | Personal Portfolio

Welcome to my personal portfolio website. This project was built to showcase my journey as a frontend developer, my skills, experience, and some of the projects I’ve worked on while learning and growing in tech.

## Built With

* HTML5
* Tailwind CSS
* JavaScript
* Lucide Icons
* SF Pro Font

## Features

* Single-page portfolio layout
* Fully responsive design for mobile and desktop
* Smooth scrolling navigation
* Modern and clean UI design
* Project showcase section
* Skills and experience sections
* Contact section for collaborations and networking

## Project Structure

* `index.html` → Main website file
* `profile.png` → Profile image used in the hero section
* `imgs/` → Folder containing other images and assets

## Running the Project

No installation or build setup is needed. Just open the `index.html` file in your browser and the website will run normally.
